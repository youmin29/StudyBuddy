## 🚑 버그 수정

### 앱 재시작 시 로그인 세션 유지 안 되는 문제 수정
- 앱을 껐다 켜면 로그인이 풀리던 문제를 수정했어요
- Supabase 세션을 `localStorage` 대신 앱 데이터 폴더(`auth.json`)에 저장하도록 변경했어요
- 로컬 서버 포트가 매번 달라지면서 `localStorage` origin이 바뀌던 근본 원인을 해결했어요

## ✨ 새 기능

### 셔플 / 다음 곡 / 반복 재생 정상 작동
- **셔플**: YouTube 플레이리스트가 로드된 경우에만 활성화돼요. 단일 영상일 땐 버튼이 비활성화됩니다
- **다음 곡**: YouTube 플레이리스트면 다음 영상으로 이동, 단일 영상이면 라이브러리의 다음 저장 항목으로 이동해요
- **반복 재생**: 플레이리스트는 전체 반복, 단일 영상은 해당 영상을 반복 재생해요
- 셔플·반복 켜짐 시 버튼이 강조 표시되어 현재 상태를 바로 확인할 수 있어요
