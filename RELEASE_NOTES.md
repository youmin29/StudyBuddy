# v1.4.1

## 🐛 버그 수정

### 개발 환경 데이터 경로 불일치 수정
- `package.json`에 `productName` 누락으로 `npm run dev` 실행 시 userData 경로가 프로덕션(`Study Buddy`)과 달라 로그인 및 로컬 데이터를 읽지 못하던 문제 수정

### 업데이트 알림 API 오류 처리 개선
- GitHub Releases가 없을 때 콘솔에 에러가 출력되던 문제 수정 (`r.ok` 체크 추가)

## 🎨 개선

### 플레이리스트 라이브러리 고정 높이 + 내부 스크롤
- 플레이리스트가 많아질수록 패널이 길어지던 문제 수정
- ResizeObserver로 왼쪽 패널 높이를 측정해 오른쪽 패널에 동일하게 적용
- 패널 높이는 타이머 탭과 동일하게 고정되고, 목록이 많으면 내부에서 스크롤
