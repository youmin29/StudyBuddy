# v1.3.0

## ✨ 새로운 기능

### 🎨 배경 테마 5종
- **핑크 🌸 · 민트 🌿 · 선셋 🌅 · 오션 🌊 · 라벤더 💜** 총 5가지 테마 지원
- 테마를 바꾸면 버튼·아이콘·슬라이더·스크롤바·텍스트 색상이 전체 연동
- 프로필 메뉴 → **테마 변경** 항목에서 현재 스와치 미리보기와 함께 전환 가능
- 선택한 테마는 앱 재시작 후에도 유지

### 👤 닉네임 편집
- 프로필 메뉴에서 이름에 마우스를 올리면 **연필 아이콘** 등장
- 클릭하면 인라인 편집 모드 — Enter로 저장, Escape로 취소
- 닉네임은 Supabase `user_metadata`에 저장되어 다기기에서 동기화

### ⚡ Supabase Realtime 실시간 동기화
- **할 일 목록**: 다른 기기에서 추가·완료·삭제 시 새로고침 없이 즉시 반영
- **재생목록**: 다른 기기에서 저장·삭제 시 자동 갱신
- 자기 기기의 변경은 에코(중복 반영) 없이 처리
- 로그아웃 시 구독 자동 해제

---

## 🎨 개선

### 전체 색상 테마 연동
- 타이머 모드 색상(집중·휴식) → CSS 변수 기반으로 전환
- 볼륨 슬라이더 thumb · 스크롤바 · 진행 바 모두 테마 색상 적용
- 캘린더 화살표·요일·날짜 텍스트 색상 테마 연동
- 투두리스트 제목·텍스트·설정 아이콘 색상 테마 연동
- 플레이리스트 라이브러리 텍스트·뱃지 색상 테마 연동
- StudyBuddy 로고 아이콘 색상 테마 연동

### 프로필 메뉴 개편
- 헤더의 별도 테마 버튼 → 프로필 메뉴 내 아코디언으로 통합
- 프로필 모달이 버튼에 더 가깝게 위치 조정

### 버그 수정
- JSX 요소에 `style` 속성 중복 선언 경고 수정

# v1.2.0

## ✨ 새로운 기능

### 🍅 뽀모도로 타이머 추가
- YouTube 플레이어 우측 패널에 **Playlist / Timer** 탭으로 통합
- 집중 / 짧은 휴식 / 긴 휴식 세 가지 모드 지원
- 4세션 완료 시 자동으로 긴 휴식 전환
- 타이머 시간 직접 설정 가능 (집중·짧은 휴식·긴 휴식 각각)
- 총 집중 시간 및 완료 세션 수 추적 + 초기화
- 타이머 종료 시 macOS 알림 발송
- 설정·통계는 앱 재시작 후에도 유지

## 🎨 개선

### UI 문구 한글화
- 주요 UI 문구를 한글로 통일
  - 뽀모도로: 집중 세션 / 짧은 휴식 / 긴 휴식 / 시작 / 일시정지 / 완료 팝업
  - 헤더: 게스트 모드 / 탭하여 로그인 / 같이 공부해요!
  - 프로필 메뉴: 동기화됨 / 클라우드 동기화 활성화
  - 로그인 모달: 클라우드 동기화 혜택 / 로그인 & 동기화
  - 플레이리스트: 재생목록 저장 / 저장 / 취소 / 즐겨찾기 / 최근

### 뽀모도로 아이콘 개선
- 긴 휴식 아이콘을 🛏️ (침대)로 변경
